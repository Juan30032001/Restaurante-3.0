# Sistema de Gestión de Restaurante
Proyecto desarrollado en *Python* aplicando *Programación Orientada a Objetos (POO)*, con estructura modular para la gestión de productos y clientes de un restaurante.

## 📋 Características
- Clase Producto con validaciones, @property y @setter
- Clase Cliente implementada con @dataclass
- Servicio centralizado en la clase Restaurante
- Menú interactivo por consola
- Operaciones: Registrar, Listar y Buscar productos y clientes

## 📂 Estructura
restaurante_app/
├── modelos/       # Definición de clases (entidades)
├── servicios/     # Lógica de negocio
└── main.py        # Ejecución y menú
